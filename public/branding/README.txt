CMCHUB Branding Assets

Use these fixed paths so you can replace branding without code changes:

- public/branding/logo.png      -> main logo used in header/sidebar
- public/branding/favicon.ico   -> browser tab icon

Recommended:
- logo.png with transparent background
- favicon.ico with 32x32 and 16x16 variants

After replacing files, restart or hard-refresh the browser.
